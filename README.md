Install Librarys

For Linux you may need to disable key ring 

```sh
Python3 -m keyring --disable
```
```sh
pip install pillow
```
```sh
python3 -m pip install image
```
```sh
sudo apt-get install python3-tk
```
```sh
sudo apt-get install python3-pil python3-pil.imagetk
```

To run GUI

```sh
cd SyncDemo
```
```sh
python3 gui.py
```
